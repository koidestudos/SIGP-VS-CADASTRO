IMAGENS PERSONALIZADAS — SIGP-VS
================================

Coloque seus arquivos nesta pasta (public/assets/custom/) com os nomes abaixo.
O sistema tentará carregar a imagem personalizada primeiro; se não existir, usa o SVG padrão.

1) Logo SESAPI
   Arquivo: logo-sesapi.png  (também aceita .jpg ou .webp)
   Usado em: barra lateral, tela de login

2) Mapa do Piauí (BI Gerencial)
   Arquivo: mapa-piaui.png  (também aceita .jpg ou .webp)
   Usado em: aba "Por Município" do BI Gerencial

Após adicionar as imagens, faça um novo deploy ou recarregue a página (Ctrl+F5).
